# Final Catalog Source

The product catalog in this build was regenerated exclusively from:

`Fleurs Catalog - All Products.csv`

Import summary:
- 175 CSV rows
- 172 master products
- 175 size/price variants
- Occasion, Arrangement Type, Collection / Series, Material Type, Pricing Type, and Order Type come from the CSV
- Products with a Collection / Series use `[Collection / Series] - [Product Name]`
- Previous catalog/demo products and category values are removed
- The three dummy bouquet images remain only as storefront fallback thumbnails

Operational schema version 22 replaces a previously saved catalog with this final catalog during hydration, while preserving the other Business OS data slices.
